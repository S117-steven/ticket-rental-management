## 版本说明

### 版本格式

```
MAJOR.MINOR.PATCH
```

- **MAJOR** - 不兼容的 API 变更
- **MINOR** - 向后兼容的功能性新增
- **PATCH** - 向后兼容的问题修正

### 更新类型

- **新增** - 新功能
- **变更** - 现有功能的变更
- **修复** - Bug 修复
- **移除** - 移除的功能
- **安全** - 安全相关的更新
